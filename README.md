**請於啟動前確保設置環境變數正確**

#

前端: REACT, NODE v18.14.0\
後端: .NET 8 \
資料庫: SQLite

#

## 啟動

前端 -> /client \
後端 -> /server

### client

```
npm install
```
```
npm run start
```
listening on: http://localhost:3000/

### server

```
dotnet run
```
listening on: http://localhost:5118/
